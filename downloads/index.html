exact

It's for modern browser. It uses ES5 and HTML5 APIs. Coding in ES6 are recommended for development.

exact-jQuery

It's for old browser. It uses jQuery to operate DOM. Besides jQuery 1.x, es5-shim and es5-sham are also required. It's important to note that some features are not supported in this edition. See here.

exact-cheerio

It's for server-side rendering. Thanks to cheerio.
